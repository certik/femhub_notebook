Welcome!

Sage is a different approach to mathematics software.

The FEMhub Notebook

With the FEMhub Notebook anyone can create, collaborate on, and publish interactive worksheets. In a worksheet, one can write code using Sage, Python, and other software included in Sage.

General and Advanced Pure and Applied Mathematics

Use FEMhub for studying calculus, elementary to very advanced number theory, cryptography, commutative algebra, group theory, graph theory, numerical and exact linear algebra, and more.

Use an Open Source Alternative

By using FEMhub you help to support a viable open source alternative to Magma, Maple, Mathematica, and MATLAB. FEMhub includes many high-quality open source math packages.

Use Most Mathematics Software from Within Sage

Sage makes it easy for you to use most mathematics software together. FEMhub includes GAP, GP/PARI, Maxima, and Singular, and dozens of other open packages.

Use a Mainstream Programming Language

You work with FEMhub using the highly regarded scripting language Python. You can write programs that combine serious mathematics with anything else.
